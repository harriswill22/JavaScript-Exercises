Do's and Donts of Javascript

Do's

  • Declare Variables
  • Use === instead of ==
  • Use Semicolons
  • Use delete properly

Dont's

  • Avoid Global Variables
  • Don't Use eval()
  • Don't Use with()
  • Avoid setting variables to undefined
  • Avoid Using new with Javascript Objects

Closure Defined

A closure is the local variables for a function - kept alive after the function has returned

Closure is when a function is able to remember to access its lexical scope even when that function is executing outside its lexical scope

** A closure is function having access to parent scope, even after the parent function has closed. (callbacks)(setTimeout)

Algorithms and Data Structures

Big O Notation

It allows us to talk formally about how the runtime of an algorithm grows as the inputs grow

O(n+10) = 0(n) O(1000n+50) = O(n) O(n^2 +5n +8) = O(n^2)

n matters the most
  • BigO for adding key into an object O(1)
  • BigO for accessing key in an object O(1)
  • BigO for removing key from an object O(1)
